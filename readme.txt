Web Site Builder 1.0.3 is a tool to have an overview for web pages, templates and page parts.
Place the templates,pages and pages part in a folder.
Call the program with the map file the only argument.(ex: wsb files.txt)
The first row is the source folder. The second row is the destination folder. See "files.txt" for the third and fourth rows.
The rest of the file is the site map. Example and information in "files.txt" .
There is also a ".wsb.log" file at the user home folder.
